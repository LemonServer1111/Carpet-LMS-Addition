# 规则

- `pearlIgnoreEntityCollision`：末影珍珠忽略实体碰撞（不传送，像空气一样穿过去）。分类：`lms`、`survival`。默认：`false`。
- `fakePlayerDropAll`：开启 `/player <name> dropall`（等同 `/player <name> dropstack all`），让假人一次性丢出背包全部物品，并支持常见的时间/频率参数。分类：`lms`、`survival`。默认：`false`。
- `fragileVaults`：把宝库的硬度和爆炸抗性降到铜箱子水平。分类：`lms`、`survival`。默认：`false`。
