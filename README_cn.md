# Carpet LMS Addition

Carpet LMS Addition 是一个小型 Carpet 扩展，待完善。

[English](README.md) | 中文

## 规则

### pearlIgnoreEntityCollision

末影珍珠忽略实体碰撞（不传送，像空气一样穿过去）。

分类：`lms`、`survival`。

默认：`false`。

### fakePlayerDropAll

开启 /player <name> dropall（等同 dropstack all）

分类：`lms`、`survival`。

默认：`false`。

### fragileVaults

把宝库的爆炸抗性降到信标水平（TNT 可炸掉）。

分类：`lms`、`survival`。

默认：`false`。

### fragileTrialSpawners

让试炼刷怪笼的爆炸抗性与普通刷怪笼一致。

分类：`lms`、`survival`。

默认：`false`。

### allayHealInterval

悦灵被动回血的 tick 间隔（原版 10，至少 1）。

分类：`lms`、`survival`。

默认：`10`。
